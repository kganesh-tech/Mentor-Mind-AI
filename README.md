# MentorMind-AI

MentorMind-AI is an AI-powered educational platform designed to help students learn in a smarter and more personalized way. The platform provides interactive learning support, AI-generated assistance, and difficulty-based question solving to improve student learning experiences.



## Features

- AI-powered learning assistance
- Beginner, Medium, and Hard difficulty levels
- Interactive question-solving support
- Personalized student guidance
- Modern and responsive dashboard UI
- Educational support system for students



## Built With

- HTML
- CSS
- JavaScript
- AI APIs
